    En tant que développeur web, participation au projet DECLADSN : (2 sur le projet) : Projet réalisé en binôme sur une année. Le projet permet de récupérer et contrôler des fichiers de DSN (Déclaration sociale nominative) fourni par d'autres applications en interne. Le contrôle se fait par un BATCH Symfony lancé chaque début de mois. Si le fichier ne contient pas d'erreurs il sera stocké en BDD(PostgreSQL). Cela permet de stocker en bases un nombre conséquent d'informations (le fichier et le mois du fichier, les individus du fichier ainsi que leur cotisations et paiements). Un deuxième BATCH SYMFONY 4 est exécuté dans un second temps. Il permet d'envoyer sur une plateforme externe (Net-Entreprises) tous les fichiers enregistrés pour le mois en cours en BDD. En milieu de mois, un troisième BATCH est exécuté afin de récupérer d'autres informations comme le taux de prélèvement à la source des individus. Informations qui sont retournées via une API au format XML, qui sont ensuite formattées pour être enregistrées en BDD. 1ere étape : Développer le BATCH de contrôle des fichiers de DSN qui permet ensuite aussi l'intégration des données du fichier en BDD. 2eme étape : Développement de l'application WEB permettant aux personnes autorisées de suivre l'état d'une déclaration (intégrée en BDD-> envoyée à Net-entreprises...) Cela permet de voir si tout le processus se passe correctement afin d'être le plus réactif possible. L'application permet aussi de voir les montants importants pour une vérification manuelle (vérification des montants de cotisations, nombre d'individus...) Cela permet aussi d'invalider une DSN afin de la réintégrer correctement. 3eme étape : Développement du BATCH qui permet de récupérer les fichiers XML générés par Net-Entreprises tous les mois grâce aux DSN envoyées. L'application est présentée sous forme de tableau de bord afin de suivre l'évolution de chaque fichier. L'application permet aussi d'autres fonctionnalités : Télécharger le fichier de DSN, Télécharger un fichier CSV permettant de vérifier les cotisations et paiements de chaque indivdu
